#Photography Portfolio Page

A simple and responsive photography portfolio page built using HTML and CSS.  
This project features a sidebar navigation menu with a hamburger toggle and social media icons, suitable for showcasing photography work or creative portfolios.

Features
- Sidebar navigation with hamburger menu  
- CSS-only toggle (no JavaScript)  
- Font Awesome icons  
- Google Fonts styling  
- Clean and modern layout  

Technologies Used
- HTML5  
- CSS3
